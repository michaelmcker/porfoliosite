Animatable SVG

Workflow map graphic.

The SVG is built as named layers: grid, labels, base routes, animated route flows, nodes, outcome modules, and orbit details. CSS handles the default draw/flow motion. JavaScript can restart or freeze it through the embedded SVG document when viewed from the local server.

CSS hooks

Animate `.route`, `.flow`, `.node`, `.node-core`, `.core-ripple`, `.traveler`, `.traveler-dot`, `.traveler-halo`, and `.orbit-dot`. The asset includes reduced-motion handling.

JS hooks

Target `#route-serp`, `#route-keywords`, `#route-brand`, `#route-proof`, `#travelers`, `#nodes`, and `#output-modules` through the object document. Use `localhost`; raw `file://` blocks this access.

Page use

The homepage and style reference both consume the same `assets/hero-workflow-map.svg` file, so animation changes stay centralized.
